Maison de Beauté - Beauty Salon Online Presence
Project Overview

Maison de Beauté is a multi-page-responsive website, created through both HTML & internal CSS, for a beauty salon. This website demonstrates the use of web principles to design a layout that is userfriendly yet has style, covers all aspects of a salon (service offerings, beauty tips, photo gallery and service menu), allows easy navigation with login/sign-up pages and a contact page.

- Front-End web development fundamentals.    This Project demonstrates basic Front-End Development concepts (layout design, responsive design, Flexbox, Animation & Cloud-hosted images).
